**Earth** is the design system for Frankcrum, providing a consistent and reusable set of UI components and patterns to accelerate development across the organization.

This repository includes all the stylesheets for the Earth package

## Installation

```bash
pnpm add @frankcrum/earth
```

## Usage

1. Import our base stylesheet _after_ the tailwind import:

```css
@import 'tailwindcss';
@import '@frankcrum/earth';
```

2. Enjoy Earth!
